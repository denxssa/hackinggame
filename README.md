#  Hacking Puzzle Game
A browser-based mini-game where players solve puzzles, decode clues, and "hack" their way through levels.  
Built with HTML, CSS,PHP and JavaScript.

---

##  Overview
This game simulates a hacker-style experience with interactive puzzles.  
Players progress by solving logic challenges, decoding hints, and interacting with UI elements styled like a hacker terminal.

---

##  Features
-  Multiple puzzle levels  
-  Hacker-style UI (green text, black background, glow effects)  
-  Interactive buttons, animations, and feedback  
-  Skill-based challenges (logic, pattern recognition, decoding)  
-  Mentally engaging and fun
